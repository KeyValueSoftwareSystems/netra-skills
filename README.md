## Skill

### 1) Netra Best Practices

File: [skills/netra-best-practices/SKILL.md](skills/netra-best-practices/SKILL.md)

Use this as the default end-to-end and code-first Netra playbook.

Highlights:
- Setup, tracing, context, metrics, evaluation, simulation, and troubleshooting in one workflow
- Code samples for integration, evaluation, and simulation
- MCP trace-analysis workflow for debugging and incident investigation

## How To Use

1. Open Copilot Chat in VS Code.
2. Invoke `netra-best-practices` and pass your context.
3. Include concrete details in your prompt:
   - language (Python or TypeScript)
   - framework/provider stack
   - desired success criteria and known failure modes
