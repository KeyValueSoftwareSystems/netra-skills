# Netra Best Practices

Use this skill as the default end-to-end guide for integrating, operating, and improving AI systems with Netra.

## Installation

Install the Netra SDK for your stack:

```bash
# Python (pip)
pip install netra-sdk

# Python (uv)
uv add netra-sdk

# TypeScript / JavaScript (npm)
npm install netra-sdk
```
